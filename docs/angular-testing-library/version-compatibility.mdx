title: Version compatibility
---

An overview of the compatibility between different versions of Angular Testing Library and Angular.

| Angular | Angular Testing Library |
| ------- | ----------------------- |
| 16.x    | 13.x \|\| 14.x          |
| >= 15.1 | 13.x \|\| 14.x          |
| < 15.1  | 11.x \|\| 12.x          |
| 14.x    | 11.x \|\| 12.x          |
