title: Version compatibility
---

An overview of the compatibility between different versions of Angular Testing
Library and Angular.

| Angular | Angular Testing Library      |
| ------- | ---------------------------- |
| 18.x    | 17.x, 16.x, 15.x, 14.x, 13.x |
| 17.x    | 17.x, 16.x, 15.x, 14.x, 13.x |
| 16.x    | 14.x, 13.x                   |
| >= 15.1 | 14.x, 13.x                   |
| < 15.1  | 12.x, 11.x                   |
| 14.x    | 12.x, 11.x                   |
