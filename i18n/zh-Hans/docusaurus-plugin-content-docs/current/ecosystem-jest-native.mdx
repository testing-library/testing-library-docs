[`Jest Native`](https://github.com/testing-library/jest-native) 是 `React Native Testing Library` 的配套库，为Jest提供自定义元素匹配器。

```bash npm2yarn
npm install --save-dev @testing-library/jest-native
```

```jsx
const {queryByTestId} = render(
  <View>
    <View testID="not-empty">
      <Text testID="empty" />
    </View>
    <Text testID="visible">Visible Example</Text>
  </View>,
)

expect(queryByTestId('not-empty')).not.toBeEmpty()
```

> 注意：当使用其中一些匹配器时，你可能需要确保你使用查询函数（如`queryByTestId`）而不是获取函数（如`getByTestId`）。
> 否则，`get*` 函数可能会在你的断言之前抛出一个错误。

请查阅[Jest Native的文档](https://github.com/testing-library/jest-native)，了解可用匹配器的完整列表。
