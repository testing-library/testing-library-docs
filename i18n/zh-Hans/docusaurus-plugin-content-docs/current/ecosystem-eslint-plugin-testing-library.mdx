[`eslint-plugin-testing-library`][gh] 是 Testing Library 的ESLint插件，
帮助用户在编写测试时遵循最佳实践并预测常见错误。

![eslint-plugin-testing-library example showing an error](/img/eslint-plugin-screenshot.png)

这个插件包括:

- 几个 Testing Library 具体代码审查的规则 用于测试库特定代码的若干规则。
- 推荐规则的可共享配置。
- 可共享的配置，用于按框架制定具体规则。Angular, React, Vue。
- 一些可自动修复的规则。

在GitHub上找到更多关于可用规则和配置的信息。

- [eslint-plugin-testing-library on GitHub][gh]
