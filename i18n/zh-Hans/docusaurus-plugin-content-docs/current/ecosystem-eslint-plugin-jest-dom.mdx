[`eslint-plugin-jest-dom`][gh] 是Jest DOM的ESLint插件，帮助用户在编写测试时遵循最佳实践并预测常见的错误。

这个插件包括:

- 几个 Jest DOM 具体代码审查规则.
- 推荐规则的可共享配置.
- 一些自动修复的规则.

在GitHub上找到更多关于可用规则和配置的信息。

- [eslint-plugin-jest-dom on GitHub][gh]

[gh]: https://github.com/testing-library/eslint-plugin-jest-dom
