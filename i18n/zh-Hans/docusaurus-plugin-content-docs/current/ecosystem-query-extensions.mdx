[`query-extensions`][gh] 是 Testing Library 的一个实验性配套库，它将一个更高层次的API混入标准的 `@testing-library` 核心查询。

```bash npm2yarn
npm install --save-dev query-extensions
```

请查看 [github repo][gh] 的文档，了解实例和原理。

[gh]: https://github.com/tjefferson08/query-extensions
